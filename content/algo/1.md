### 1.2 유용한 ASCII Code 값
'A'~'Z' 코드 값 범위 : 65 ~ 90 \
'a'~'z' 코드 값 범위 : 97 ~ 122 \
'0'~'9' 코드 값 범위 : 48 ~ 57

[**example**] \
char c = "9"울 숫자로 변환하려면 (c-'0')
